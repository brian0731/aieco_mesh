Place offline Hong Kong XYZ tiles here using this layout:

  {z}/{x}/{y}.png

The bundled tiles cover zoom levels 10 through 16. The map can enlarge these
tiles up to zoom 18. Run tool/import_hk_tiles.ps1 to
validate and copy a licensed/exported XYZ tile directory into this folder.

Zoom 17 is additionally bundled for Kowloon and the Wong Tai Sin area only
(114.12-114.27 E, 22.29-22.39 N).

Zoom 18 is bundled only for Wong Tai Sin District
(114.18-114.24 E, 22.32-22.36 N).
